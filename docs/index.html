Commands by Category

Utility

`ping` - Show bot latency
`uptime` - Show bot uptime
`userinfo` - Info about a user
`botinfo` - Show bot stats and details

Integrations

`github [repo]` - Show latest GitHub event
`mcstatus` - Show Minecraft server status

Moderation

`kick`, `ban`, `unban`, `warn`, `mute`, `unmute`, `purge`, `softban`

Admin

`setprefix`, `lockdown`, `slowmode`, `clear`

Developer

`reload` - Reload bot extensions

Server Commands (Mainly used if linked with an MC server)

'serverban' - Adds a role named "Server Banned" to a user | Prevents them from seeing the in-game server-chat from Discord
'serverunban' - Removes the Server Banned role
'exile' - Sends a user to the Shadow Realm | Prevents the user from seeing any channel
'forgive' - Resolves the user of all their sins
'appban' - Bans a user from submitting an application forum post
