Commands

Utility Commands

  • avatar — Displays the given user's avatar
  • ping — Shows the bot latency
  • userinfo — Displays information about a provided user. Defaults to you if none provided.
  • uptime — Shows how long the bot has been online
  • serverinfo — Shows information about the current Discord server
  • setprefix — Change the bot prefix for this server (Admin only)
  • roleinfo — Gives information about a given role
  • botinfo — Displays information about the bot
  • ascii — Turns any text into ASCII
  • say — Repeats the given message as the bot
  • eightball — We all know about the "magic 8-ball"

Moderation Commands

  • ban — Bans a user from the server
  • unban — Unbans a user
  • mute — Mutes a user
  • unmute — Unmutes a user
  • clear — Deletes a specified number of messages

Server Commands

These are mainly for if you run a Minecraft server with this bot, these come in handy for easy permissions handling

  • serverban — Adds the "Server Banned" role to a user
  • serverunban — Removes the "Server Banned" role from a user
  • exile — Moves user to "Exiled" role
  • forgive — Removes user from "Exiled" role
  • staffadd — Adds a user to the Admin list (Currently only supports the "Admin" role for starting out on)
  • promote — Promotes the added Admin to Senior Admin
  • senior_suspend — Removes a rogue Senior Admin
  • suspend — Does exactly what senior_suspend does but for normal Admins

Admin Commands

  • lockdown — Locks down the channel that the command was executed in
  • unlock — Unlocks the locked channel
  • setprefix — Changes bot prefix (also listed in Utility)
  • stop — Shutdown the bot (Developer only)
  • restart — Restarts the bot (Developer only)
  • reload — Reloads the given cog (Refer to the "cogs" section)

Integration Commands

  • github — Shows latest GitHub repo event (also in Utility)
  • fact — Shows a useless fact
  • weather — Shows the current forecast of an inputted location
  • mcstatus — Gives a status report of a given Minecraft server address.

Timezone Commands

  • settimezone — Sets your personal timezone
  • timefor — Shows the current time in your set timezone
  • convert — Shows the conversion between one timezone and another
